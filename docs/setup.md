# Setup Notes

This phase contains architecture and folder scaffolding only. Runtime code will be added in later phases.

## Prerequisites Planned For Later Phases

- Node.js LTS
- pnpm
- PostgreSQL
- Docker and Docker Compose
- Google Cloud project with Gmail API enabled
- Google Cloud project with Google Sheets API enabled
- OpenAI API key

Never commit real secrets.
